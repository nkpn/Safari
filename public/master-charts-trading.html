Master Charts Trading —

Helping novices navigate the rocky path from a novice to a consistently profitable trader.

MasterchartsTrading.com — was launched with the primary goal of helping novices navigate the often-rocky path from a novice to a consistently profitable trader.

I had to create a clean and functional website where information about the trading techniques and tools was easily digestible by novice traders. I started with revising the copy for the entire site, and created a set of clear instructions of how to use the trading instruments.
