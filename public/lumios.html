Lumios —

The mental health iOS App that motivates people and provides emotional support.

It gives the user affirmation that he/she is loved and appreciated. LUMIOS delivers to users expertly crafted, motivational content via push notifications, text, and static images wrapped in a cutting-edge user-centric design.

  • Product Management
  • Branding
  • Website Design

How it works

The App consists of a selection of static images with text on them. The app can be configured by the user to send push notifications, that when opened, take the user directly to the app.

After the splash screen and onboarding. After the user can jump straight to content or create an account. In the selection of a specific theme, the user is shown a screen with his/her first content: a static image with a motivational text on it.

User can

  • Pick one of several topics: family, career, relationship, change, etc
  • Change the backgrounds
  • Change the text and font
  • Share an image to social media
  • Choose the time and frequency of notifications
  • Write a note in the built-in journal
  • Perform a breathing exercise
